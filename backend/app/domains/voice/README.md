# Voice

The voice domain extracts M-group speech descriptions from audio raw sources, including timbre, pitch, pace, pauses, rhythm, and accent.

Describing these features is not voice cloning. Imitation or generated voice output is a separate workflow that requires explicit authorization and stricter source-attribution checks.

中文：Voice 记录音色、音高、语速、停顿、节奏和口音等 M 组特征。这不等于声音克隆；拟声或生成声音必须走独立授权和片段归属流程。
